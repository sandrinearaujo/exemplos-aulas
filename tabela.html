Tags funcionalidades
head delimita o cabeçalho do documento.
html é responsável que faz a ligação entre um documento e outro,
body representa o corpo do documento.
meta permite inserir metadados ao seu documento
style Permite incluir elementos CSS dentro da tag;
header definem um cabeçalho
nav define um conteúdo de navegação
h1 Título de maior valor hierárquico
h6 Título de menor valor hierárquico
p compõe um parágrafo;
b Transforma o conteúdo em negrito;
div Define uma divisão da página.
title Define o título do elemento;
lang Define o idioma principal do elemento;
class Atribui uma classe ao elemento
